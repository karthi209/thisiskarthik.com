Meta

This site is just HTML and CSS compiled into static pages. I wrote a blog generator in Go because I got tired of platforms that change their terms of service every Tuesday and frameworks whose dependencies break every Friday.

A full build takes 2 milliseconds. That's longer than zero, but shorter than most modern websites take to load.

Almost everything lives in the static build. Few moving parts means fewer things to break at 2am.

Design Philosophy

The site has a subtle Commodore 64 aesthetic. IBM Plex Mono everywhere. Cyan accents. ASCII box-drawing characters. Dark theme. Zero JavaScript.

It's designed to be readable, fast, and still work in 20 years when everything else has broken.

Typography

IBM Plex Mono for everything. It's monospace, but actually readable. Good for both UI elements and body text. Gives that retro computing feel without sacrificing legibility.

How It Works

Custom static site generator in Go. Goldmark for Markdown. YAML for frontmatter. No build tools. No dependencies that will break next week.

The code is simple and readable. No clever abstractions that will confuse future me.

Colors

Background: #1a1a2e (dark but not black)

Text: #eaeaea (off-white, easy on the eyes)

Accent: #00d9ff (cyan, very C64)
