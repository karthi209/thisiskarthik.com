Colophon

This site is written in old-school HTML and CSS and assembled as static pages. It relies on no database, no runtime server, and no third-party frameworks. Minimal and classic. A full build of this site completes in {{.BuildTime}} milliseconds on this machine. These figures reflect the state of the site at the time of writing.

Typography

Headings and titles use IM Fell Great Primer, an open-source revival of early eighteenth-century type. Designed by Igino Marini, the face draws from the work of the seventeenth-century Dutch punchcutter Dirck Voskens, whose letters later found their way into the presses of William Caslon. Body text uses Libertinus Serif for comfortable long-form reading. Both typefaces are open source and licensed under the SIL Open Font License.

Colour and Aesthetic

The palette is deliberately restrained: a vintage-ish paper (#fbf9f4) for the background, a slightly worn ink (#2b2723) for the text, and a muted rubrication red (#C62828) for headings and occasional emphasis.

Illustration

The title artwork is a heraldic illustration. Heraldry favours symbolism over likeness, using formal shapes and limited colour to suggest identity rather than describe it.

Technical Details

The site is statically compiled using my own blog engine written in Go, using the standard library and two small external dependencies: Goldmark for Markdown parsing and YAML for frontmatter. No build tools are employed beyond the compiler itself. The code is kept simple and readable, with the intention that it may be understood, reused, or altered without ceremony.

Philosophy

The tone of the site is intentionally restrained. Clarity is preferred over novelty and permanence over convenience.
What you read is what was written.
