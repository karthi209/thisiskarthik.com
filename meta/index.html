Meta

This site is just HTML and CSS compiled into static pages using a blog generator I wrote in Go because I got tired of platforms that change their terms of service every Tuesday. A full build of this site takes 2 milliseconds which is longer than zero but shorter than most modern websites take to load, and almost everything lives in the static build which means few moving parts and fewer things to break.

Typography

Bricolage Grotesk for everything because it's a modern geometric sans-serif that's clean and readable and has that indie aesthetic without being too weird, with system fonts as fallback for older browsers. Body text uses regular weight (400) for comfortable reading, while headings use semibold (600) for better distinction.

How It Works

Custom static site generator in Go with Goldmark for Markdown and YAML for frontmatter and no build tools and no dependencies that will break next week, and the code is simple and readable with no clever abstractions that will confuse future me.

Colors

Background is #1e1e1e which is a muted dark gray that's easy on the eyes for long reading sessions, and text is #e8e8e8 which is off-white for good contrast without being harsh, and the accent is #00d9ff which is a bright cyan for links and highlights that stands out without being too saturated.

Source Code

It's all on GitHub if you're into that sort of thing.

Performance

Fonts are loaded with preconnect hints and display=swap to minimize layout shifts, and the site uses semantic HTML and minimal CSS for fast loading and good performance across all devices.
